# Assets

Postman collection and related resources for testing the URL Shortener API.

## Files

- **postman-collection.json** - Complete API collection with all endpoints
- **postman-environment.json** - Environment variables for local/production
- **POSTMAN.md** - Postman setup and usage guide

## Quick Start

1. Import collection: `assets/postman-collection.json`
2. Import environment: `assets/postman-environment.json`
3. Select environment (Local/Production)
4. Run requests

See [POSTMAN.md](POSTMAN.md) for detailed instructions.
